# Portfolio Website Guide

Welcome to your new portfolio website! This project is built using modern HTML, CSS, and JavaScript.

## 🚀 How to Open the Website

It varies depending on how you want to view it, but the easiest way is:

1. **Directly in Browser (Easiest Method):**
   - Go to your Desktop.
   - Open the folder named `My portfolio`.
   - Find the file named `index.html`.
   - **Double-click** on `index.html`. It will automatically open your portfolio in your default web browser (like Google Chrome, Microsoft Edge, or Firefox).

2. **Using a Code Editor (For editing):**
   - If you have VS Code installed, right-click inside the `My portfolio` folder and select "Open with Code".
   - You can install an extension called **Live Server**.
   - Right-click on `index.html` inside VS Code and select **"Open with Live Server"**. This will open it in your browser and automatically refresh when you make changes.

## 📁 Project Structure

- `index.html`: The main structure and content of your website.
- `style.css`: All the colors, animations, and dark-theme designs.
- `script.js`: The logic for animations and scrolling.

Enjoy your new premium portfolio!
